Project Name:

Lilac

Graduate Research Assistant

University of Texas Arlington


A Tor like circuit based low latency anonymous light weight instant messaging communication architecture. This architecture demonstrates anonymity in context with unlinkability and unobservability for the secured real-time communication.

The system uses a relay based anonymity mechanism where circuits are built and routed over a set of trusted nodes in the network. Unlike other typical light weight communication systems, Lilac uses user-pseudonymity and ephemeral message exchange, which eliminates conventional user registration and storing the messages for the future retrieval.

This system is deployed to be used for end-to-end communication. It does not require user end configuration and modification; thus it provides a hands-on ease of access on both desktop and mobile environments acquiring the anonymity by reasonable tradeoff amongst the deployability, compatibility, usability and efficiency.

Technologies used: nodejs, Javascript, Selenium.
This project is sponsored by National Science Foundation award no. CNS- 0954133.
